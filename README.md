# To-Do List App

A simple To-Do List application built using HTML, CSS, and JavaScript.
This project focuses on DOM manipulation, event handling, and storing tasks using localStorage.

## Features

Add new tasks

Mark tasks as completed

Delete tasks

Automatically saves tasks using localStorage

Fully responsive and beginner-friendly

## Project Structure
index.html      → Structure  
style.css       → Styling  
script.js       → Logic (DOM, events, storage)

## How It Works

User enters a task and clicks Add

Task appears in the list

Checkbox allows marking tasks as completed

Delete button removes task

All tasks are saved and loaded from localStorage
